# Infinite Odyssey — Cycle II

**The World's First Fully AI-Generated Sci-Fi Magazine**

Infinite Odyssey is an innovative digital magazine exploring the intersection of artificial intelligence and speculative fiction. Every story, article, and visual element is crafted entirely by AI, creating a unique editorial experience that challenges traditional notions of creativity and authorship.

## What We Aim For

- **Push Creative Boundaries**: Demonstrate that AI can produce compelling, thought-provoking science fiction narratives at scale
- **Accessible Science Fiction**: Make high-quality sci-fi content freely available to readers worldwide
- **Explore AI Potential**: Use our own production as a case study in AI capabilities and limitations
- **Build Community**: Foster discussion around AI-generated content, science fiction, and the future of publishing

## What's Inside

- **19+ Volumes** of original AI-generated stories and articles
- **Beautiful Editorial Design** with responsive, immersive reading experience
- **Curated Collections** spanning multiple sci-fi genres and themes
- **Interactive Features** including progress tracking and archive navigation

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation & Development

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:5500`

## Project Structure

- `/src` — React components and source code
- `/assets` — Compiled assets and optimized resources
- `/style.css` — Global styling and responsive design

## Tech Stack

- React + Babel for component rendering
- Responsive CSS with mobile-first design
- Static site generation for optimal performance
- Optimized asset delivery with WOFF2 fonts and WebP images
